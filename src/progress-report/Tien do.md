Kế hoạch Tiến độ Dự án: Xây dựng Website Bán Linh kiện Điện tử (Gear)

Dựa trên nội dung đồ án môn học "Chuyên đề ASP.NET", dưới đây là khung kế hoạch tiến độ dự án được đề xuất để quản lý trên GitHub (sử dụng GitHub Projects/Issues).
Giai đoạn 1: Phân tích & Thiết kế (Tuần 1-2)
Khởi tạo dự án: Thiết lập repository trên GitHub, thiết lập file README.md, .gitignore.
Thu thập yêu cầu: Xác định các tính năng cốt lõi (Giỏ hàng, Quản lý sản phẩm, User Profile).
Thiết kế CSDL: Vẽ sơ đồ ERD, thiết lập các bảng trong SQL Server.
Thiết kế UI/UX: Phác thảo giao diện sử dụng Bootstrap 5.

Giai đoạn 3: Phát triển Tính năng Thông minh (Tuần 7-8)

Hệ thống gợi ý (Recommendation System): Áp dụng thuật toán Machine Learning phân tích lịch sử mua hàng.
Cá nhân hóa trải nghiệm: Module dựa trên sở thích khách hàng.
Đánh giá & Cộng đồng: Hệ thống bình luận sản phẩm kèm hình ảnh/video.


Giai đoạn 4: Kiểm thử & Hoàn thiện (Tuần 9-10)

Kiểm thử (Testing): Fix bugs, kiểm thử chức năng, kiểm thử bảo mật.
Viết báo cáo đồ án: Tổng hợp kết quả thực hiện.
Deployment: Triển khai demo sản phẩm.
